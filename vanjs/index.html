VanJS

VanJS is a meetup for web developers downtown Vancouver. The focus is on Javascript, HTML5, and the open web. Javascript frameworks, browser functionality, tips and tricks are all popular topics. The meeting format is two 45 minute talks, followed by pub time for discussion.
